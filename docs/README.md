# GoText Documentation

The documentation at the root of this folder (`architecture/`, `guides/`, `howto/`,
`ai_agent_rules/`, `screenshots/`, `appicon.png`) always describes the **current** app — right
now that's the **v3** redesign.

Earlier generations of the app's documentation are preserved as frozen snapshots, not updated
going forward:

- [`v1_docs/`](v1_docs/) — original Text Processing Suite / early GoText docs.
- [`v2_docs/`](v2_docs/) — pre-v3-redesign docs and UI screenshots.

`ai_agent_rules/` is not versioned per app generation — it's a living set of coding standards
for AI coding agents that evolves independently of the app's doc history.

The working material used while drafting the v3 redesign (spec drafts, prompt drafts, early
mockups) has been removed now that it's superseded by the finalized docs at the root.
